# 长治文旅大生态服务平台

[![codecov](https://codecov.io/gh/likedanni/ccthub/branch/ci/migration-jwt-test/graph/badge.svg)](https://codecov.io/gh/likedanni/ccthub)

## 项目简介

本项目为“长治文旅大生态服务平台”统一工程仓库，
用于承载文档、前端、后端、数据库、接口、运维、合规及 AI 协作规范。

## 目录结构说明

- docs/ 产品需求、业务流程、架构设计（核心）
- frontend/ 前端工程（小程序 + Web）
- backend/ 后端微服务
- database/ 数据模型与 SQL
- api/ API 设计与接口文档
- ops/ 运维、SLA、部署方案
- compliance/ 法务与合规文档
- ai/ AI 开发约束与 Prompt

## 使用规范（强烈推荐）

- 严禁 AI 跨目录修改文件
- 一次只喂一个目录给 AI
- 文档即事实标准
