# 长治文旅大生态服务平台 - 管理后台

基于 Vue 3 + Vite + Element Plus 开发的管理后台系统

## 技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- Element Plus
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
