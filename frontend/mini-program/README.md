# 长治文旅大生态服务平台 - 微信小程序

基于 uni-app 开发的微信小程序用户端

## 技术栈

- uni-app
- Vue 3
- 微信小程序

## 开发

```bash
# 安装依赖
npm install

# 开发模式(微信小程序)
npm run dev:mp-weixin

# 构建(微信小程序)
npm run build:mp-weixin
```

## 目录结构

```
mini-program/
├── pages/          # 页面
│   ├── index/      # 首页
│   └── user/       # 个人中心
├── api/            # API 接口
├── utils/          # 工具函数
├── static/         # 静态资源
├── App.vue         # 应用配置
└── pages.json      # 页面配置
```

## 注意事项

1. 需要在微信开发者工具中导入此项目
2. 需要配置合法的服务器域名
3. 本地开发时可以在开发者工具中勾选"不校验合法域名"
